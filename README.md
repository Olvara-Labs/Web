# Olvara Labs - Web

Premium, high-fidelity web experiences for Olvara Labs.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Three.js (@react-three/fiber, @react-three/drei)

## Features
- **3D Interactive Hero**: A high-fidelity WebGL render of the Olvara Labs brand identity.
- **Fluid Transitions**: Bespoke circular reveal page transitions based on user click coordinates.
- **Responsive Design**: Mobile-first, premium "Apple-like" aesthetic.

## Development
```bash
npm install
npm run dev
```
